About

Hi! I am a game developer and fencing coach from England, my passion for game development started when I was 11 creating games in scratch, which then led me to dabble in other game engines such as Stencly and Game Maker. Eventually after discovering programming language such as python and java at the age of 15 I started learning Unity and released my first game Range Shooter at the age of 16. My most recent projects have all been using either Unreal Engine or creating games from scratch in C++ and DirectX 11. 

On this webpage you can see my current projects and some of my previous projects, there is a mix of university projects and personal projects, my two favourite projects being my game engine and vehicle sim (my biggest and most difficult project). I love the sense of achievement you get when creating games and software, especially when something is so difficult that it seems impossible but with perseverance it ends up working well. I also enjoy 3D modeling assets for my games, however, I am no where near being an artist which is evident from this website.

I am currently studying a masters in game development at the University of Central Lancashire, soon to be graduating this year (2023) and I cannot wait to finally step foot in industry.

Masters Game Engine Project

  •  University 
  •  Masters 
  •  In Progress 

ABOUT

This project is my university masters project. The aim is to create an game engine that supports Physics and C# Scripting from scratch in C++ and the use of libraries.  The main graphics API in use is DirectX 11, I have implemented this with a static class and abstraction too allow other APIs to be implemented at a later date.

Features

  • Physics
  • C# Scritping
  • Removable Editor
  • Entity Component System(ECS)
  • Scene Serialisation
  • Rendering Abstraction

Libraries

  • PhysX 5.1
  • Mono
  • ImGui
  • EnTT
  • GLM
  • ASSIMP
  • Spdlog
  • DirectXTK (DirectX 11)
  • SDL
  • Yaml-cpp

Computer Space Clone

  •  Personal 
  •  Completed 

ABOUT

In my foundation year of university I had to do a presentation of my choice, I chose to do it on the history of video games. This inspired me to recreate Computer Space in unity, which took less than week to do. Computer Space is a clone of the first video game arcade machine developed in 1971 a year before the popular game pong was released. Computer space was developed by Nutting Associates and Atari, inc.
